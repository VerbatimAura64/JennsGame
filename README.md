This new game idea project is something I started working on during the Glitch City Strawberry Jam on 3/8/26. The goal of the project is to begin reconstructing popular snap and build city games with resource production and management, citizen retainment and utilization, as well as a personalized style aimed to keep my girlfriend's attention while learning how to use mobile ads as well. It's going to take a while since my surface pro isn't really meant for game design and construction, but the scripting is the fun part. Most of this project will and should probably be handled by the desktop, but the ability to start and the use of touchscreen on my tablet help emulate settings similar to those of other tablets and handheld devices.
